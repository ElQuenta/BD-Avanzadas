pasos
- Crear Contenedores
```bash
docker-compose up -d
```
- Ejecutar `init.sql`
- Descargar Dependencias
```bash
npm i
```
- Ejecutar Proyecto
```bash
npm run dev
```

>[!IMPORTANT]
>No se requiere .env

>[!NOTE]
>No se almacena el token
